El problema

Lo mismo, cada turno, a mano.

Copiar datos, dar formato, redactar el correo y enviarlo. Una tarea repetitiva que consume tiempo, falla y depende de que una persona esté ese día.

⏱️

~120 min perdidos

Tiempo de alguien cualificado en una tarea mecánica.

⚠️

Errores de copy-paste

Un dato mal pegado y el reporte sale equivocado.

📭

No escala

Si esa persona falta, no hay reporte. Atado a una persona.

Cómo funciona

Un flujo, sin manos.

Cuatro pasos automáticos desde tu archivo hasta la bandeja de entrada del responsable.

1

Lee

CSV o Excel. Detecta columnas y métricas numéricas.

2

Agrupa

Por área, línea u operario. Suma y totaliza.

3

Genera

Excel (resumen + detalle) y HTML limpio.

4

Envía

Por SMTP, a la hora exacta. Cero pasos manuales.

Tecnología

Estándar, ligero y portable.

Sin frameworks pesados. Funciona en cualquier máquina y se empaqueta en un ejecutable para quien no tenga Python.

Python 3.11+ openpyxl CLI (argparse) SMTP + STARTTLS PyInstaller → .exe GitHub Actions CI pytest

Uso · un comando
# Reporte agrupando por área
shiftreport datos.csv --group-by area

# Programar el envío a las 14:30
shiftreport datos.xlsx \
  --group-by linea \
  --email-to jefe@empresa.com \
  --at 14:30

# Salida: outputs/*.xlsx + *.html

Quédate con la tarde libre.

Genérico, open source y listo para usar. Por si a tu equipo le ahorra esas dos horas de cada turno.
